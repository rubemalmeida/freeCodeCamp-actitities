Value properties

These global properties return a simple value; they have no properties or methods.

  • Infinity
  • NaN
  • undefined
  • null literal

The global Infinity property is a numeric value representing infinity. Eg.

var maxNumber = Math.pow(10, 1000); // max positive number

if (maxNumber === Infinity) {
console.log("Let's call it Infinity!");
// expected output: "Let's call it Infinity!"
}

console.log(1 / maxNumber);
// expected output: 0
Function properties

These global functions—functions which are called globally rather than on an object—directly return their results to the caller.

  • eval()
  • uneval() - not standardized
  • isFinite()
  • isNaN()
  • parseFloat()
  • parseInt()
  • decodeURI()
  • decodeURIComponent()
  • encodeURI()
  • encodeURIComponent()
  • escape() - deprecated
  • unescape() - deprecated

The eval() function evaluates JavaScript code represented as a string. Eg.

console.log(eval('2 + 2'));
// expected output: 4

console.log(eval(new String('2 + 2')));
// expected output: 2 + 2

console.log(eval('2 + 2') === eval('4'));
// expected output: true

console.log(eval('2 + 2') === eval(new String('2 + 2')));
// expected output: false
Fundamental objects

These are the fundamental, basic objects upon which all other objects are based. This includes objects that represent general objects, functions, and errors.

  • Object
  • Function
  • Boolean
  • Symbol
  • Error
  • EvalError
  • InternalError
  • RangeError
  • ReferenceError
  • SyntaxError
  • TypeError
  • URIError

The Object constructor creates an object wrapper. Eg.

// Object initialiser or literal
{ [ nameValuePair1[, nameValuePair2[, ...nameValuePairN] ] ] }

// Called as a constructor
new Object([value])
Numbers and dates

These are the base objects representing numbers, dates, and mathematical calculations.

  • Number
  • Math
  • Date

The Number JavaScript object is a wrapper object allowing you to work with numerical values. A Number object is created using the Number() constructor. Eg.

new Number(value);
